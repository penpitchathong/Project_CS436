วิธีการใช้งาน
บันทึกโค้ด:คัดลอกโค้ดทั้งหมดที่คุณให้มา และบันทึกเป็นไฟล์ชื่อ bmi_tracker.py (หรือชื่อใดก็ได้ที่ลงท้ายด้วย.py)

อินเทอร์เฟซผู้ใช้:
บันทึกข้อมูลใหม่: ป้อน ชื่อ (Name), น้ำหนัก (Weight) เป็นกิโลกรัม, และ ส่วนสูง (Height) เป็นเซนติเมตร
คลิกปุ่ม "Calculate & Save BMI" เพื่อคำนวณและบันทึกข้อมูลลงในฐานข้อมูล
ดูประวัติ: ข้อมูลประวัติล่าสุดจะแสดงอยู่ในกล่องข้อความด้านล่าง พร้อม ID สำหรับใช้ในการลบ
ลบข้อมูลประวัติ: ป้อน Record ID ที่ต้องการลบ (ดูจากตารางประวัติ) ลงในช่อง "Record ID ที่ต้องการลบ" แล้วคลิกปุ่ม "Delete Record"
